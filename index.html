Project A.N.G.E.L - 3D Roguelike

During a 184-hour summer internship, I participated in a 6-person team to develop A.N.G.E.L, a 3D roguelike game about a robot fighting a rogue artificial intelligence. I was responsible for implementing movement mechanics, programming robot parts and their attacks, implementing animations, and designing and coding the boss room. Game was made in C# using Unity Engine.

Description

A.N.G.E.L is a 3D roguelike game in which players control a robot navigating the floors of a scrapyard. Players can fight enemies, take control of hostile robots, and upgrade their own systems. The game features procedurally designed levels, challenging combat mechanics, and unique boss encounters.
